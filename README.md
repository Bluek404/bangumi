# bangumi
新番放送时间表，命令行版

## 使用
直接运行显示当天时间表

加参数`all`显示全部

```
$ bangumi
┌──────────────────────────────┐
│             周日             │
├─────┬────────────────────────┤
│08:00│亚尔斯兰战记            │
│08:00│龙珠超                  │
│08:00│噬神者                  │
│21:00│VENUS PROJECT －CLIMAX－│
│21:27│和歌子酒                │
└─────┴────────────────────────┘
```

**数据来自: <http://acgdb.com>**
